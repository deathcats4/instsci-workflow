# Attribution and License Notice

This repository is a modified public beta build based on the MIT-licensed InstSci project.

Original copyright remains with the InstSci contributors. The original MIT license is preserved in `LICENSE` and applies to the included source unless a file states otherwise.

This modified build focuses on:

- public-package cleanup and privacy hygiene;
- OA-first and visible-browser evidence workflow documentation;
- `file_status` / `standard_status` / `result_evidence` status contract clarity;
- Zotero handoff and linked-PDF workflow guidance;
- public beta review packaging and verification commands.

InstSci does not bypass publisher or institutional access controls. Closed-access retrieval requires the user's own legal institutional access, and SSO, CAPTCHA, 2FA, or WAF checks must be completed by the user in a visible browser.
